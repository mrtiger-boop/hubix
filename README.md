# Hubix v2.0 — Voisin Ruby

Version complète centrée sur le système de rencontre aléatoire.

## Installation
Ouvre `index.html` dans ton navigateur.

## Intégré
- Thème rouge/rubis Hubix
- Connexion / création de compte locale
- Profil complet
- Recherche aléatoire poussée
- Filtres : sexe, âge, objectif, pays, langue, type de discussion, statut, intérêts
- Animation de recherche
- Match avec profil complet
- Chat privé local
- Ajouter ami / favori / passer / bloquer / signaler
- Chat mondial par langue
- Boutique avec cadres de profil
- Coins, achat et équipement de cadres
- Amis, favoris, bloqués
- Paramètres
