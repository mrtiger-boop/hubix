# Hubix v5.0.0 — Full Supabase

Version complète propre de Hubix.

## Objectif
- Auth Supabase
- Profils Supabase
- Présence en ligne Supabase
- Chat mondial Supabase Realtime
- Match aléatoire avec vrais profils connectés
- Chat privé Supabase Realtime
- Amis / favoris / blocages Supabase
- Notifications prêtes
- PWA prête

## Installation
1. Mets tes clés dans `js/services/supabase-config.js`.
2. Lance `supabase/schema_v5.sql` dans Supabase SQL Editor.
3. Active Realtime sur `presence`, `world_messages`, `private_messages`, `notifications`, `friend_requests`.
4. Lance avec Live Server.
5. Push sur GitHub pour Vercel.
