# Installation Hubix v5

1. Remplace ton projet actuel par cette version.
2. Ouvre `js/services/supabase-config.js` et colle tes clés Supabase.
3. Lance `supabase/schema_v5.sql` dans Supabase SQL Editor.
4. Active Realtime sur `presence`, `world_messages`, `private_messages`, `notifications`, `friend_requests`.
5. Teste avec deux comptes sur deux navigateurs/appareils.
